# Livro Caixa - Clínica

<p align="justify">
Descreve a associação entre livros caixa e clínicas. Este modelo permite que um mesmo livro caixa seja usado por uma ou mais clínicas e mantém a chave de relacionamento com o livro caixa separado do cadastro do caixa.
</p>
